# 🛡️ Bug Bounty Automation Tool

A custom automation tool that mimics real-world bug bounty workflows, designed around the **OWASP Top 10**.

## 🔍 Features
- Automated scanning for:
  - XSS, SQLi, XXE, Insecure Deserialization, and more
- Modular Python scripts for each vulnerability type
- Easily extendable for new tests and integrations

## 🚀 Technologies
- Python
- `requests`, `BeautifulSoup`, `re`, etc.
- Custom logic for each OWASP Top 10 vulnerability
